# Full Stack Plugin

全栈开发流程插件，提供从原型图到 PRD 文档的完整支持。

## 功能特性

### Skills

- **prototype-prd**: 从图片或原型图中提取需求和交互逻辑，生成结构化 PRD 文档

## 目录结构

```
full-stack-plugin/
├── .claude-plugin/
│   └── plugin.json    # 插件配置
├── skills/
│   └── prototype-prd/
│       └── SKILL.md   # PRD 生成技能
└── README.md
```

## 安装

通过 Claude Code 插件市场安装
